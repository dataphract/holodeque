# Changelog

## [0.2.0] - 2021-07-08
### Added
- `clear()` and `truncate()`
- `as_slices()` and `as_mut_slices()`
- Quickcheck tests for `ArrayDeque`

## [0.1.0] - 2021-07-06
### Added
- First version of the crate!
- `SliceDeque` and `ArrayDeque`
- `#![no_std]` support
- Optional `serde` support for both types
